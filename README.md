## Installation of dependencies

### With plain pip

```
pip install -r requirements.txt
```

### With uv

```
uv pip install -r requirements 
```

## Run the detection

Put your images in a directory called `./images/`, then run

```
python fish_detection.py`
```

or  

```
uv python fish_detection.py
```
